# 训练输出命名规范

更新时间：2026-01-23

## 1. 日志命名
路径：`d:\Graduation_Project\logs\`

命名格式：
```
<stage>_<model>_<date>_<run_id>.log
```

示例：
```
sft_qwen_20260123_run01.log
grpo_qwen_20260123_run01.log
```

## 2. 模型保存命名
路径：`d:\Graduation_Project\models\`

命名格式：
```
<stage>_<model>_<date>_<run_id>
```

示例：
```
sft_qwen_20260123_run01
grpo_qwen_20260123_run01
```

## 3. 数据快照命名
路径：`d:\Graduation_Project\output\`

命名格式：
```
<dataset>_<date>_<version>.jsonl
```

示例：
```
llm_inputs_20260123_v4.jsonl
```
