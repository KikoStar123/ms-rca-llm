# 展示层字段映射（JSON -> 表格）

更新时间：2026-01-23

## 1. JSON 字段
来自统一输出 Schema（`输出JSON_Schema.md`）：
- `event_id`
- `time`
- `top_candidates`
- `prediction.root_cause_component`
- `prediction.fault_type`
- `prediction.kpi`
- `prediction.related_container`
- `prediction.explanation`

## 2. 表格列映射建议
| 表格列 | JSON 字段 |
| --- | --- |
| 事件ID | event_id |
| 时间 | time |
| 根因组件 | prediction.root_cause_component |
| 故障类型 | prediction.fault_type |
| KPI | prediction.kpi |
| 容器 | prediction.related_container |
| Top‑K | top_candidates |
| 解释摘要 | prediction.explanation |

## 3. 显示建议
- `Top‑K` 建议以逗号拼接展示
- `解释摘要` 可折叠展示，避免页面过长
