# GRPO 训练配置模板（仅配置说明）

更新时间：2026-01-23

## 1. 输入与输出
- 基础模型：SFT 后的权重
- 训练数据：`d:\Graduation_Project\output\llm_inputs_v4.jsonl`
- 输出格式：结构化 JSON（字段同 SFT）

## 2. GRPO 训练要点（示意）
- 采样数量：`G=2~4`（每个样本的候选输出数）
- KL 约束系数：`β=0.001~0.01`
- clip：`ϵ=0.1~0.2`
- 学习率：`1e-5`
- max_seq_len：`2048`
- completion_len：`512~1024`

## 3. 奖励设计（建议）
**总奖励 = α * 正确性 + β * 一致性 + γ * 格式合规**
- 正确性：根因 + 故障类型准确
- 一致性：解释与证据匹配（引用 evidence）
- 格式合规：JSON 字段齐全、无多余文本

## 4. 评估建议
- 结构化输出可解析率
- 与 SFT 结果对比的一致性
- Top‑K 纠错能力（在候选中选对）
