# 实验结论段落（可直接引用）

更新时间：2026-01-23

## 1. 小模型候选基线结论
实验结果表明，纯频次根因排序在 Top‑K 命中率上表现不稳定；引入加权策略后，hit@1 与 hit@3 命中率获得稳定提升。最优基线配置（trace_weight=1.5, anomaly_weight=0.8, anomaly_once=true）在 Top‑1 上进一步提升，说明在多源观测下合理的权重分配能够提高候选根因的排序质量。

## 2. 大模型输入分布与结构化输出
构建的 LLM 输入样本规模为 81 条，覆盖网络延迟、CPU 故障、网络丢包与数据库异常等多类故障类型，类别分布相对均衡。结构化输出经统一 Schema 校验，合规率达到 100%，表明该格式能够稳定承载系统输出并支持后续展示与评估。

## 3. 端到端流程验证
端到端小样例流程（解压→SFT 样本→LLM 输入→输出校验）完整跑通，验证了数据准备、候选生成与输出格式化的可复现性，为后续模型训练与系统落地提供可靠基础。
